# Patoka

Patoka is a collection of Jupyter Lab widgets for IBM Qiskit

## User-side installation

`pip install patoka`

## Usage

Refer to notebooks in the `demos` directory.

## Developers

### Vertual environment

Run `source bin/activate`

### Install libraries

1. For Python:
 `pip install -e .`

2. For Node:
 `npm i`

### Bug report

Please use issues.
